# Week 1 - Day 1

## 📝 Overview

Day 1 focused on understanding the fundamentals of SQL and practicing basic database operations using DB Fiddle.  
The session mainly covered writing queries, filtering data, sorting records, and understanding how relational databases work.

This day helped in building the foundation required for advanced SQL concepts and Data Engineering workflows.

---

# 🎯 Objectives

- Understand SQL basics
- Practice writing simple queries
- Learn filtering and sorting techniques
- Work with aggregate functions
- Understand database table relationships
- Practice query execution using DB Fiddle
