# Week 1 - Day 1

## 📝 Overview

Day 1 focused on understanding the fundamentals of SQL and practicing basic database operations using DB Fiddle.  
The session mainly covered writing queries, filtering records, sorting data, and understanding relational database concepts.

This day helped in building a strong foundation for advanced SQL concepts and future Data Engineering topics.
