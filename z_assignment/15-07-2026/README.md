# SQL Joins Practice Hub

---

Welcome to the **SQL Joins Practice Hub**. This repository is dedicated to mastering SQL JOIN operations through structured practice assignments and real-world relational database scenarios.

The exercises are designed to strengthen the understanding of how multiple tables interact and how meaningful information can be extracted by combining related datasets.

---


## Repository Overview

This repository covers different types of SQL JOIN operations, starting from the fundamentals and progressing to advanced query challenges.

Topics are organized into multiple difficulty levels to encourage step-by-step learning and practical problem-solving.

---


## Join Types Covered

### INNER JOIN

Retrieve only the records that have matching values in both tables.

**Practice Includes:**

- Employee and department relationships
- Employee project assignments
- Department-wise employee details
- Multi-table joins
- Aggregate queries using INNER JOIN

---


### LEFT JOIN

Retrieve all records from the left table along with matching records from the right table.

**Practice Includes:**

- Employees without departments
- Employees without assigned projects
- Departments with employee counts
- Projects with or without assigned employees

---


### RIGHT JOIN

Retrieve all records from the right table along with matching records from the left table.

**Practice Includes:**

- Departments without employees
- Projects without assignments
- Department and project analysis
- Employee count reports

---


### FULL OUTER JOIN

Retrieve all matching and non-matching records from both tables.

**Practice Includes:**

- Unmatched employees and departments
- Department and project comparisons
- NULL handling
- Complete relationship analysis

---


### Anti Joins

Identify records that do not have matching values in related tables.

**Topics Covered:**

- LEFT ANTI JOIN
- RIGHT ANTI JOIN
- FULL ANTI JOIN

**Practice Includes:**

- Employees without departments
- Departments without employees
- Projects without departments
- Orphan record identification
